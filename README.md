# ECOSAVE

Smart simulation-based system for real-time home electricity monitoring.

ECOSAVE is a simulation-based smart energy monitoring system that allows users to track the electricity consumption of household appliances in real-time through a web interface. Using current sensors, an ESP8266 Wi-Fi module, and Firebase, the system collects and analyzes data, providing clear visual feedback and consumption predictions to help users reduce their energy bills.

## Features
- Real-time electricity monitoring per device
- Safe simulation using Tinkercad
- Live data storage in Firebase
- Interactive dashboard built with Laravel
- Predictive billing using regression analysis

## Tech Stack
- Arduino (Tinkercad Simulation)
- ACS712 Current Sensor
- ESP8266 Wi-Fi Module
- Firebase Realtime Database
- Laravel (Backend)
- HTML, CSS, JavaScript (Frontend)
